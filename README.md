## 🚀 快速开始

### ⚡ 一键启动（推荐）

**Windows用户：**
```bash
# 智能启动器（推荐）
start_videogenius.bat

# 快速启动（如果已安装依赖）
quick_start.bat

# 首次安装
setup_environment.bat
```

**手动安装：**
```bash
# 1. 克隆项目（仅10.89MB）
git clone https://github.com/your-repo/VideoGenius.git
cd VideoGenius

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
python -m streamlit run webui/Main.py
```

## ✨ 核心功能

### 🎬 专业级视频制作
- **智能脚本生成** - AI自动生成高质量视频脚本
